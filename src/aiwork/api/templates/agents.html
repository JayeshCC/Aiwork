How Agents Work

Specialized Skills

Each agent has a specific role, goal, and backstory that guides their behavior and decision-making.

Equipped with Tools

Agents use specialized tools like OpenVINO adapters, calculators, and database connectors.

Memory & Context

Agents can remember past interactions and use context to make better decisions.

Example: Creating an Agent 

from aiwork.core.agent import Agent
from aiwork.core.memory import VectorMemory

# Create memory
memory = VectorMemory()
memory.add("User prefers detailed explanations")

# Create agent
analyst = Agent(
    role="Financial Analyst",
    goal="Detect fraud in transactions",
    backstory="20 years of experience in fraud detection",
    tools=[calculator_tool, database_tool],
    memory=memory,
    verbose=True
)
